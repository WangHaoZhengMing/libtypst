# libtypst (Typst C Bridge)

A C-API bridge for the [Typst](https://typst.app/) typesetting system, written in Rust. This library enables you to embed the full Typst compiler directly into C/C++, Swift, Python, Go, and theoretically any language that supports C FFI.

## Features

- Complete C ABI for embedding the `typst` compiler.
- **System Fonts Support**: Automatically loads system fonts and Typst's default fonts.
- **Package Manager**: Intercepts requests like `#import "@preview/cetz:0.4.2"` and automatically downloads/caches official packages, seamlessly mimicking the real Typst CLI environment.
- Render Typst snippets directly to a PDF file or an in-memory byte buffer.

## How to Build

Requirement: You need to have [Rust (cargo)](https://rustup.rs/) installed.

### Standard Build (macOS / Linux)

```bash
cargo build --release
```

The resulting libraries will be dumped in the `target/release/` directory as `libtypst_c.a` (Static) and `libtypst_c.dylib`/`libtypst_c.so` (Dynamic). 

### macOS Universal Build (Intel + Apple Silicon)

If you want to package a library that is suitable for all Mac target architectures:

```bash
./scripts/build_mac.sh
```

This will output Universal binaries into a new `universal-macos/` directory using `lipo`.

## Quick Start (C Example)

You can compile and run the provided C example:

**macOS:**
```bash
clang examples/example.c -o example \
  -I include \
  target/release/libtypst_c.a \
  -framework Security -framework CoreFoundation

./example
```

**Linux:**
```bash
gcc examples/example.c -o example \
  -I include \
  -L target/release -ltypst_c \
  -lm -ldl -lpthread

./example
```
